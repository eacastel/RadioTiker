RadioTiker vNext

Thin Agent downloads for device onboarding (no full repo clone needed).

  • vNext Linux x86_64: radiotiker-thin-agent-vnext-latest-linux
  • vNext onboard helper script: radiotiker-vnext-onboard-latest.sh
  • API health: /streamer/api/health
Linux onboarding path A (click links in browser)
  1. Click both download links above.
  2. Open terminal and run:
cd ~/Downloads
chmod +x radiotiker-thin-agent-vnext-latest-linux radiotiker-vnext-onboard-latest.sh
mv radiotiker-thin-agent-vnext-latest-linux radiotiker-vnext-onboard-latest.sh ~/
cd ~
ENABLE_ACOUSTID_SCAN=1 ./radiotiker-vnext-onboard-latest.sh --user-id <user> --library-path "<path>" --gui
Linux onboarding path B (copy/paste with curl)
cd ~
curl -fLO https://radio.tiker.es/downloads/radiotiker-thin-agent-vnext-latest-linux
curl -fLO https://radio.tiker.es/downloads/radiotiker-vnext-onboard-latest.sh
chmod +x radiotiker-thin-agent-vnext-latest-linux radiotiker-vnext-onboard-latest.sh
ENABLE_ACOUSTID_SCAN=1 ./radiotiker-vnext-onboard-latest.sh --user-id <user> --library-path "<path>" --gui
Notes
  • --gui launches the Tk GUI after onboarding; --run launches headless mode.
  • ENABLE_ACOUSTID_SCAN=1 enables fingerprint collection when fpcalc is available.
  • For very large libraries, batching/retry is automatic; tracks will trickle into player/library while scan continues.
